NAME URL VERB DESCRIPTION METHOD
Index /blogs GET Display a list of all blogs Blog.find()
New /blogs/new GET Displays form to make a new blog N/A
Create /blogs POST Add new blog to db and redirectes to another page Blog.create()
Show /blogs/:id GET Show the information of a particular blog Blog.findById()
Edit /blogs/:id/edit GET Show edit form for a particular blog Blog.findById()
Update /blogs/:id PUT Updates a particular blog and redirects to another page Blog.findByIdAndUpdate()
Delete /blogs/:id DELETE Deletes a particular blog and redirects to another page Blog.findByIdAndRemove()
